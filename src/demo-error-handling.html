🚨 Global Error Handling System

This demo showcases the global error handling system with modal popups and backdrop. Click the buttons below to trigger different types of errors and see how the system handles them.

How it works:

  • All errors are caught by the global error handler
  • Error details are displayed in a modal with backdrop
  • Detailed stack traces are available for debugging
  • Errors are logged to the console with additional context
  • Custom error metadata is preserved and displayed
